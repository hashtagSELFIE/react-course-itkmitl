# Config
## Eslint
### Install eslint config to project
```
$ yarn add --dev eslint-config-nss
```
### Create `.eslintrc` and add config
```
{
  "extends": "nss/node"
}
```
### Enable `eslint.autoFixOnSave` in VSCode setting
